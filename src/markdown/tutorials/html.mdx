# HTML5 Basics

## The anatomy of a page

## Part 1: The unseen

The parts of a page you cannot see include:

- [DocType](#DocType)
- Comments
- HTML Tags
- Head Section
- Style Tags
- Script Tags

### DocType
Tells the browser the type of document to expect. In HTML5, simply put 'html':
```
<!DOCTYPE html> 
```
For older versions of HTML, [more info was needed](https://www.w3schools.com/tags/tag_doctype.asp)

### Comments
Comments allow you to have words in HTML that do not run as code.
#### Example:
```
<!-- This is a comment -->
```


### HTML Tags
#### Example:
 ```
 <!--Sets the language to English and text direction to "Left to Right"-->

 <html lang="en" dir="ltr"> ... </html>
 ```
